  • Теоретические задания
    1. Число уравнений треугольной системы называется ее рангом (rank). Под рангом
    произвольной системы линейных уравнений понимают ранг эквивалентной ей
    треугольной системы.
    2. Наименьшее число линейных уравнений, которое необходимо добавить к системе
    линейных уравнений над A с тем, чтобы она имела единственное решение в поле
    частных кольца A, называют размерностью множества решений этой системы.
    3. За множеством решений системы алгебраических уравнений закрепилось название
    многообразие, за множеством решений линейной системы — линейное
    многообразие. Многообразие размерности 1 называют линией, размерности 2 —
    поверхностью, линейные многообразия размерности 1 называют прямой линией,
    размерности 2 — плоскостью.
    4. 1) Приводим систему к треугольной форме по алгоритму Гаусса из функции
    triangulation.
    2) Если уравнений меньше, чем переменных, тогда приравниваем число
    переменных к номеру ранга системы или любому случайному значению
    3) Можно решать систему в общем виде, тогда решения уравнения будут содержать
    переменные
    4) Решаем треугольную форму с помощью функции “tsolve”
